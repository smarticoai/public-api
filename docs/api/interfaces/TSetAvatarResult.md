# Interface: TSetAvatarResult

Result of `_smartico.api.setAvatar()`.

## Properties

### err\_code

> **err\_code**: `number`

Error code. `0` = success. See `setAvatar` TSDoc for the full table.

***

### err\_message?

> `optional` **err\_message?**: `string`

Optional error message; populated on non-zero `err_code`.
